# InfiniteCanvas ## About `InfiniteCanvas` makes any HTML `<canvas>` infinite in size, meaning that you can pan it in any direction and zoom out as far as you like, just like you would any Google Map. All it takes is one line of code: // assuming canvas is an HTMLCanvasElement var infCanvas = new InfiniteCanvas(canvas); That's all!
