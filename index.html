# InfiniteCanvas ## About `InfiniteCanvas` makes a `<canvas>` element infinite in size, so that you can pan the canvas in any direction and zoom in or out as far as you like. Simply create an `InfiniteCanvas` like this: var canvasElement = document.getElementById("canvas"); var infCanvas = new InfiniteCanvas(canvasElement); And then get a `CanvasRenderingContext2D` from it like you would with a regular `<canvas>`: var context = infCanvas.getContext("2d"); That's all!
