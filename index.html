# InfiniteCanvas ## About `InfiniteCanvas` makes any `<canvas>` element virtually infinite in size. Meaning that you can pan it in any direction and zoom out as far as you like, just like you would any Google Map. All it takes is a few lines of code: var infCanvas = new InfiniteCanvas(canvas); // assuming canvas is an HTMLCanvasElement var ctx = infCanvas.getContext("2d"); That's all! This `ctx` is now a `CanvasRenderingContext2D`, which you can use to draw as described [here](https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D). Here's a live example:
var infCanvas = new InfiniteCanvas(canvas); var ctx = infCanvas.getContext("2d"); // Set line width ctx.lineWidth = 10; // Wall ctx.strokeRect(75, 140, 150, 110); // Door ctx.fillRect(130, 190, 40, 60); // Roof ctx.beginPath(); ctx.moveTo(50, 140); ctx.lineTo(150, 60); ctx.lineTo(250, 140); ctx.closePath(); ctx.stroke();
